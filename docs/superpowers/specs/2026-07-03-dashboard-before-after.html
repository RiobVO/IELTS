P1 · мотивация

«6.0 band to go» — пропасть → ближайшая веха

Band 2 при цели 8 сейчас показывает подпись «6.0 band to go» — честно, но это марафон без станций: новичок видит пропасть, а не путь. Milestone-фрейминг: шкала получает фиолетовый тик ближайшей вехи (+0.5), подпись говорит про неё, target остаётся тёмным тиком дальше. Код уже «value-aware» (низкий band рисуется тихо) — это следующий шаг той же логики. Потыкай переключатель: веха работает на любом уровне.

P3 · полировка

Мелочи одним проходом

До
• Band-шкала — только визуал: скринридеру достаётся голая подпись, без «2 out of 9, target 8».
• Заголовки тестов с префиксом «IELTS Reading Test - …» — в приложении каждый заголовок начинается одинаково, глаз ищет отличия в хвосте.
• «15 missed / 5 missed / 4 missed» — три mono-пилюли одинакового веса; счёт и так стоит рядом (0/15).
• Тултип дня недели — hover-only: с клавиатуры и на таче недостижим (aria-label есть — ок, но sighted-keyboard пользователь его не видит).
• «Welcome back» + «Hi, ultra-test» + дата — три строки на приветствие при плотном экране ниже.
После
• role="img" aria-label="Band 2 of 9, target 8, next stop 2.5" на контейнер шкалы.
• Титул чистим на рендере: title.replace(/^IELTS (Reading|Listening)( Test)?\s*[-—]\s*/,"") — категория и так в мета-строке.
• «missed»-пилюля остаётся только у худшего типа (красная смысловая), у остальных счёт справа уже всё говорит.
• Столбцам недели — tabindex="0" + показ тултипа на :focus-visible (CSS уже есть, добавить селектор).
• Приветствие в одну строку: «Hi, ultra-test · Friday, July 3» — экономим 40px над hero.
Бонус · концепт

Today's focus → Today's plan: чек-лист вместо лозунга

Самая сильная версия hero — не «один тип», а план дня из 2–3 шагов, сгенерированный из weak-списка и незаконченных попыток: дрилл слабейшего → короткая серия по №2 → review вчерашних ошибок. Чек-лист даёт то, чего лозунг не даёт: ощущение завершаемости дня (0/3 → 3/3) — главный крючок ежедневного возврата. Данные все уже на сервере, LLM не нужен — это правила. Потыкай шаги:
